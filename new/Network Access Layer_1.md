# 진행 방식

- OSI에서 1~2 계층의 경우 분리되어 있지만, 현재 대부분은 Ethernet이라는 하나의 기술로 동작한다.
- 따라서 다음과 같은 순서로 진행한다.
- 1. Ethernet이 뭔지 알아본다. 구체적으론, OSI 1~2 계층에 해당되는 어떤 기능을 제공하는지.
- 2. 1~2 계층의 역할을 알아본다. 
    - 1. 기초적인 해당 계층의 역할을 알아본다.
    - 2. 이더넷에서 어떤 구체적으로 동작하는지 알아본다.
    - 3. 이더넷이 아닌 기술을 간략하게 확인해본다.

# Ethernet

- 현대에는 이더넷(Ethernet)이 거의 모든 곳에서 쓰인다. 
- 1, 2 Layer에 포함되며, TCPIP 모델의 경우 Network Access Layer에 해당된다.
- IEEE에서 표준 관리

- Ethernet의 역할
    - 1계층에 해당
    - 2계층에 해당

- 대표 기술
    - IEEE 유선 무선 그런거. 이더넛에 포함되는 기술

- 아래 다뤄지는 내용은 이더넷을 기반으로 설명함.
- 그 이전 기술도

# 1 Layer 

## 기초

- 역할
- 핵심 기술
- 대표적인 표준
    - IEEE 유선, 무선, 블루투스, 광랜, USB 등

## 이더넷 - 딥하게

- IEEE 
    - 이더넷 유선 버전 별 속도 등
    - 이더넷 무선 별 버전 속도 등.
    - 구체적으로 어떤 걸 하는지?
- RJ45나 USB의 표준 정의는 어디서 되는지?

## 기타 기술

- 이더넷 이전에 사용되거나, 현실에서 잘 쓰이지 않는 것.

# 2 Layer

## 기초

- 역할
- 핵심 기술
- 대표적인 프로토콜

## 이더넷 - 딥하게

- 이더넷 프레임 헤더와 구조 역할
- Ethernet에서 주로 다뤄지는 내용이므로 키워드만 정리
- MAC 프로토콜, 에러 검출, 프름 제어, 프레임 생성, 주소화, 에러 복구(유선 MAC 프로토콜)
